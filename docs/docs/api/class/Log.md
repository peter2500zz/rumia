# Log

!!! info

    本文部分内容由 AI 辅助编写，如果发现错误请提交 issue。

`Log` 是 Rumia 提供的用于记录日志的类，它包括一些记录不同级别日志的静态函数。

`Log` 的字段不需要传入自身，也就是不需要 `:` 调用， `.` 调用即可。

## 字段

以下是 `Log` 所有的字段:

---

### info

记录信息级别的日志。

函数签名:

```lua
fun(format: string, ...: any)
```

参数:

- `format` string: 日志消息模板。支持普通文本或包含 printf 风格占位符（如 %s, %d, %.2f）的格式化字符串。
- 动态参数列表（可选）。用于填充 format 中的占位符。如果 format 中不包含占位符，这些参数将被转为字符串并以空格分隔追加在消息后。
