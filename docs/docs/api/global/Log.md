# Log

`Log` 是 Rumia 提供的用于记录日志的类，它包括一些记录不同级别日志的静态方法。

`Log` 的方法不需要传入自身，也就是不需要 `:` 调用， `.` 调用即可。

以下是 `Log` 提供的方法:

### info

记录信息级别的日志。

函数签名:

```lua
fun(...: any)
```

参数:

- 任意可打印内容。

### warn

记录警告级别的日志。

函数签名:

```lua
fun(...: any)
```

参数:

- 任意可打印内容。

### error

记录错误级别的日志。

函数签名:

```lua
fun(...: any)
```

参数:

- 任意可打印内容。

### debug


记录调试级别的日志。

函数签名:

```lua
fun(...: any)
```

参数:

- 任意可打印内容。

### trace

记录追踪级别的日志。

函数签名:

```lua
fun(...: any)
```

参数:

- 任意可打印内容。
