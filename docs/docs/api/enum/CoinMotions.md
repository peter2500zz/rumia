# CoinMotions

定义游戏内掉落物的运动方式，它们全部是 [CoinMotion](../alias/CoinMotion.md) 类型。

以下是全部可用枚举值:

---

### DROP_FROM_XY

从坐标落下

---

### SLOW_DROP_FROM_XY

从坐标缓慢落下

---

### POP_FROM_BACK

从后方跳出

---

### FAST_POP_FROM_BACK

从后方快速跳出

---

### COLLECT_IMMEDIATELY

直接收集

---

### AUTO_COLLECT_LATER

稍后自动收集

---

### POP_FROM_RIGHT

从屏幕右侧蹦出

---

### SPAWN_IN_SEED_SLOT

在卡槽栏生成
