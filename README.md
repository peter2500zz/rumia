本项目允许你使用 Lua 脚本为游戏《植物大战僵尸》v1.0.0.1051 编写 Mod。对于不想在运行游戏时产生设置和存档干扰的玩家，配置文件也支持隔离你的存档和游戏设置！

本仓库不提供任何有关《植物大战僵尸》v1.0.0.1051 版本的下载。

欢迎 issue 和 pr ！
