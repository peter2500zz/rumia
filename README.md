本项目允许你使用 Lua 脚本为游戏《植物大战僵尸》v1.0.0.1051 编写 Mod。  
This project allows you to write mods for *Plants vs. Zombies* v1.0.0.1051 using Lua scripts.

对于不想在运行游戏时产生设置和存档干扰的玩家，配置文件也支持隔离你的存档和游戏设置！  
For players who want to avoid interfering with existing saves and settings, the configuration file also supports isolating game data and preferences.

本仓库不提供任何有关《植物大战僵尸》v1.0.0.1051 版本的下载。  
This repository does **not** provide any downloads of *Plants vs. Zombies* v1.0.0.1051.

欢迎 issue 和 PR！  
Issues and pull requests are welcome!
